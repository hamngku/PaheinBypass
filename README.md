# Pahe.in Bypass Download Link
https://pahe.li/

Just some regular tools to make it easier to get through the download links on pahe.in.

# Features
- Bypass Download Link to Direct Link

# Installation
```
git clone https://github.com/hamngku/PaheinBypass
cd PaheinBypass
npm i
npm start
```
